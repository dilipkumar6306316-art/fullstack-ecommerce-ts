# fullstack-ecommerce
A production-grade e-commerce platform built with React, TypeScript and Node.js featuring scalable architecture and reusable components.

## Features
- User Authentication (Login / Register)
- Product Listing & Product Details
- Search Products
- Add to Cart & Wishlist
- Checkout Page
- User Profile
- Seller Dashboard (Basic)
- Admin Dashboard (Basic)

## Tech Stack
- React.js
- JavaScript (ES6)
- HTML
- CSS
- React Router

## Installation
git clone https://github.com/your-username/andrometakart.git  
cd  fullstack-ecommerce
npm install  
npm start  

## Purpose
Ye project React concepts, component structure aur e-commerce workflow ko practice karne ke liye banaya gaya hai.
